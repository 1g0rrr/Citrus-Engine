Classpublic class Emitter
InheritanceEmitter CitrusObject Object

An emitter creates particles at a specified rate with specified distribution properties. You can set the emitter's x and y location at any time as well as change the graphic of the particles that the emitter makes.

graphicproperty 
graphic:*

The graphic that will be generated for each particle. This works just like the CitrusObject's view property. The value can be 1) The path to an external image, 2) A DisplayObject class (not an instance) in String notation (view: "com.graphics.myParticle") or 3) A DisplayObject class (not an instance) in Object notation (view: MyParticle). See the documentation for ISpriteView.view for more info.
