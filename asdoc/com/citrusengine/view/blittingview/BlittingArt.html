registrationproperty 
public var registration:String = center

Values can be "center" or "topLeft". Specifies the registration point for your art. Often, static images such as backgrounds are easier to place with "topLeft" registration, while physics objects are commonly represented with "center" registration.

Constructor Detail
BlittingArt()Constructor
public function BlittingArt(defaultGraphic:Class = null)

This is the graphical representation of your CitrusObject when using the Blitting view. a BlittingArt object should contain one or more AnimationSequences. If your object is does not need to animate (such as a background), you can simply pass in a class that creates your graphic. The class that you pass in must create a BitmapData object.

Parameters
defaultGraphic:Class (default = null) — For objects without aimation, you can pass in a single BitmapData class. Useful for creating backgrounds.
