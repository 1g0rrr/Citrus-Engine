Classpublic class AnimationSequence
InheritanceAnimationSequence starling.display.Sprite

The Animation Sequence class represents all object animations in one sprite sheet. You have to create your texture atlas in your state class. Example : var hero:Hero = new Hero("Hero", {x:400, width:60, height:130, view:new AnimationSequence(textureAtlas, ["walk", "duck", "idle", "jump"], "idle")});
